EUROFLEX – Responsive Product Landing Page

This project is a responsive landing page built using HTML, CSS, and JavaScript,
based on the provided Figma design.

Features:
- Sticky header navigation
- Product image carousel with zoom effect
- Responsive layout for desktop, tablet, and mobile
- Clean and well-structured code
- Semantic HTML and reusable CSS

How to run:
Open index.html in any modern web browser.
